# Outreach Proof

The Melsoft brief requires genuine proof that the company was contacted.

Before submission:

1. Send the message in `outreach-message.txt` by email, contact form, LinkedIn, or direct message.
2. Take a screenshot or save a PDF that clearly shows the sent message and date.
3. Add that file to this folder.
4. Commit and push the evidence to GitHub.

Do not submit the draft by itself as proof. The proof must show that outreach actually happened.
